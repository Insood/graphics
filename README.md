## A collection of software rendering examples

Based on the "introduction to computer graphics" youtube video series.

### examples\00_basic_lighting

Demonstrates line drawing, mesh rendering, flat shading, barycentric shading, Phong face lighting, Phong vertex lighting, Phong-Gourand shading,Phong shading.

![00_examples](https://github.com/Insood/graphics/blob/main/images/00_combo.png?raw=true)

### examples\01_2d_transforms

Demonstrates basic 2D transforms (rotate, translate, scale) from model -> view -> device (using orthographic projection) -> screen coordinates

![01_examples](https://github.com/Insood/graphics/blob/main/images/01_gear_scene.gif?raw=true)
